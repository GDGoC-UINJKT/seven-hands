## Seven Hands

Pada 2009, sebuah tim internal di divisi keamanan jaringan membangun proyek rahasia bernama SEVEN HANDS. Tujuannya sederhana: menyimpan bukti intrusi tanpa menarik perhatian siapa pun.

Namun proyek itu gagal. Tim dibubarkan, server dipensiunkan, dan hampir semua arsip dianggap hilang.

Tujuh belas tahun kemudian, satu file muncul kembali dari backup yang seharusnya telah rusak. Tidak ada ekstensi yang jelas. Tidak ada metadata yang berguna. Hanya satu catatan kecil di log forensik:

“If you found this, it means they never reached the core.”

Analis yang menemukan file itu curiga bahwa ini bukan sekadar arsip rusak. Struktur binernya menunjukkan file itu dibungkus berulang kali dengan format yang berbeda. Bukan kesalahan. Bukan kebetulan. Ini seperti setiap orang yang pernah memegang file itu sengaja meninggalkan satu lapisan, satu era, satu jejak.
